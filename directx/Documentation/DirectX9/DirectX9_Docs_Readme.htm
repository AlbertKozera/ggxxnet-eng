PRINTING ISSUES

If you are trying to print a heading and all of its subtopics, there are several known issues.

  • The pages will have no formatting.
  • If you are using Microsoft Internet Explorer 5.5, content may be missing from your printed pages. Use Internet Explorer 5.0 or 6.0 instead. The recommended printing method is to print one topic at a time. The printed page should then contain all the formatting and content.

PIX Command-Line Options

The following command-line options are available to modify behavior of the PIX tool.

  • Launch PIX and open a .PIXRun log file:
    • PIXWin.exe aaa.PIXRun [-playstandalone]
    Optional Command Description
    -playstandalone Plays back in a window a captured Direct3D stream.

  • Launch PIX and open a .PIXexp experiment file:
    • PIXWin.exe aaa.PIXExp [-targetpath path] [-targetstartfolder folder] [-targetargs args] [-start]
    Optional Command Description
    -targetpath path Overrides the target program path with path.
    -targetstartfolder folder Overrides the target start folder with folder.
    -targetargs args Overrides target command-line arguments with args.
    -start Starts running the experiment.

  • Launch PIX and open an image file in the Image Compare View:
    • PIXWin.exe aaa.png [bbb.png] [-appenddiffmeasure path]
    Optional Command Description
    bbb.png Also opens file bbb.png for comparison with file aaa.png.
    -appenddiffmeasure path Compares files aaa.png and bbb.png, and appends a numerical measurement of the difference between the images to the text file at the specified path.

DIRECTSOUND

No DirectSound Debugging

Support for debugging DirectSound has been removed from this SDK Update.

DIRECTSHOW

No DirectShow Debug DLL

No DirectShow Debug DLL ships with this SDK release, so when you run DirectSetup, only the DirectShow Retail installation option is available.
